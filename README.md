# ぽんこつSVGジェネレーター

## 概要

マウス操作で簡単な図形（四角形、楕円、直線）やテキストを描き、SVGファイルとして保存できるツールです。

## 使い方

- **ツールの選択**: ツールバーの「長方形」「楕円」「線」「テキスト」ボタンから、描画したいものを選択します。
- **図形の描画**: キャンバス上でマウスをドラッグして、選択したツールで図形を描画します。
- **テキストの追加**: テキストツールを選択後、キャンバスをクリックすると入力ダイアログが開きます。入力後「OK」でテキストが追加されます。
- **テキストの編集**: 作成したテキストをダブルクリックすると、再度入力ダイアログで編集できます。
- **図形の選択**: 作成した図形やテキストをクリックすると選択できます。もう一度キャンバスをクリックすると選択解除されます。
- **図形の移動**: 選択した図形をドラッグアンドドロップすることで、位置を移動できます。キャンバス外にマウスが出ても追従します。
- **図形の回転**: 選択した図形の四隅（青いガイドリングが表示されます）の少し外側をドラッグすることで回転できます。`Shift`キーを押しながら回転させると、15度刻みでスナップします。
- **図形の拡大縮小 (リサイズ)**: 選択した図形の四隅の青いガイドリングの内側（リサイズハンドル）をドラッグすることでサイズを変更できます。`Shift`キーを押しながらドラッグすると、アスペクト比（縦横比）を維持して拡大縮小します（※テキスト図形は非対応、線は端点移動になります）。
- **コンテキストメニュー (重なり順変更)**: 図形を右クリックするとメニューが表示され、「最前面へ移動」「最背面へ移動」「削除」の操作を行えます。
- **キャンバスのリサイズ**: 初回ロード時にキャンバスのサイズ（幅と高さ）を設定できます。
- **図形の削除**: 作成した図形を選択し、`Delete` または `Backspace` キーを押すか、右クリックメニューから削除できます。
- **元に戻す (Undo)**: 「Undo」ボタンを押すか、ショートカットキーで操作を一つ前に戻します。回転や移動などの操作も記録・復元されます（UIの選択状態などの一時的な変化は履歴に影響しません）。
- **やり直す (Redo)**: 「Redo」ボタンを押すか、ショートカットキーで元に戻した操作をやり直します。
- **全消去**: 「クリア」ボタンを押すと、キャンバス上のすべての図形が消去されます。
- **エクスポート**: 「エクスポート」ボタンを押すと、描画した内容がSVGファイルとしてダウンロードされます。

### キーボードショートカット

| 操作             | Windows/Linux     | macOS               |
| ---------------- | ----------------- | ------------------- |
| 元に戻す (Undo)  | `Ctrl + Z`        | `Cmd + Z`           |
| やり直す (Redo)  | `Ctrl + Y`        | `Cmd + Shift + Z` |
| 選択中の図形を削除 | `Delete`, `Backspace` | `Delete`, `Backspace` |

## 開発者向けセットアップ

1.  **依存関係のインストール**:
    ```bash
    pnpm install
    pnpm exec playwright install
    ```

2.  **開発サーバーの起動**:
    - **通常モード**（デバッグ機能OFF）:
      ```bash
      pnpm run dev
      ```
    - **デバッグモード**（ログ出力と状態ビューアON）:
      ```bash
      pnpm run dev:debug
      ```
      ※ `VITE_DEBUG_MODE=true` 環境変数が読み込まれ、キャンバスの下部に現在の State Tree や Undo/Redo 履歴のログを表示する状態ビューアが表示されます。

3.  **その他のコマンド**:
    - **Lint (修正含む)**:
      ```bash
      pnpm run lint
      ```
    - **型チェック**:
      ```bash
      pnpm run typecheck
      ```
    - **テスト実行**:
      ```bash
      pnpm test
      ```
